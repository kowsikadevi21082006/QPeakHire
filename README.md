# QPeakHire

QPeakHire is a modern AI recruitment platform landing page UI designed to showcase intelligent hiring, candidate matching, and recruitment automation solutions.

## Tech Stack

* React
* Tailwind CSS
* Framer Motion

## Live Demo

## Features

* Modern SaaS Landing Page
* Responsive Design
* Smooth Animations
* AI Recruitment-Themed UI
* Glassmorphism Effects
* Interactive Components
